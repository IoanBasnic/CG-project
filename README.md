# CG-project
## What is it?
It's a WebGL application, showing a planet with asteroids. You can change the preset settings using the GUI provided in menu bar.

## Technologies
* HTML5
* CSS3
* Javascript - Using three.js, datGui.js, OrbitControls.js
* WebGL
